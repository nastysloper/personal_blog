Apache Web Server

Apache web server comes bundled with Mac OS. First step is to add yourself to the configuration file.

edit /etc/apache2/users/USERNAME.conf and add the following lines:

Options Indexes Multiviews
AllowOverride AuthConfig Limit
Order allow,deny
Allow from all
</Directory>

also make this change in http.conf:
# User home directories
510 Include /private/etc/apache2/extra/httpd-userdir.conf

note: don't edit the index.html file in /Library/Webserver/Documents your files will go in your home directory under Sites/
To make that happen, you edit the /etc/apache2/users configuration file
and the httpd-userdir.conf file.

start the server with this command:

sudo apachectl start

or
sudo apachectl restart
